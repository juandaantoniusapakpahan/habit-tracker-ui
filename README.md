# Habit Tracker & Money Management UI

Ini adalah project **React.js** untuk **Daily Habit Tracker** dan **Money Management**. Project ini dibangun menggunakan **Vite + React** dan menampilkan UI modern, responsif, dan interaktif.

### 1. Daily Habit Tracker

- Checklist untuk setiap task berdasarkan tanggal
- Support **check/uncheck** langsung via API
- Tampilan seperti **habit tracker di Excel**
- Optimistic UI update: perubahan checklist langsung tampil
- Responsif untuk desktop, tablet, dan mobile
- Tanggal disusun dari **hari ini sampai tanggal 1 bulan berjalan**

### 2. Money Management
- Tabel pemasukan dan pengeluaran bulanan
- Warna berbeda untuk **Income (hijau)** dan **Expense (merah)**
- Total balance ditampilkan di atas tabel
- Responsif di semua ukuran layar
- Data dummy random untuk testing UI

### 3. Navbar & Footer
- Navbar dengan dua menu: **Daily Tracker** dan **Money Management**
- Footer tetap di bawah halaman
- Modern dan minimalis
- Setiap halaman tetap berada di tengah viewport
